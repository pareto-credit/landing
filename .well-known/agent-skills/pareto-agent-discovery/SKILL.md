# Pareto Agent Discovery

Use this skill when an agent needs to discover Pareto public pages, developer documentation, or API metadata.

## Resources

- Homepage: https://pareto.credit/
- Sitemap: https://pareto.credit/sitemap.xml
- Robots policy: https://pareto.credit/robots.txt
- API catalog: https://pareto.credit/.well-known/api-catalog
- OpenAPI description: https://pareto.credit/openapi.json
- API documentation: https://docs.pareto.credit/developers/api

## Notes

The Pareto API uses Bearer API keys. Request access through the API documentation before calling protected endpoints.
